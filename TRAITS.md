## Mutations / Mewtations

16 Mutation Pairs (16 x 2 = 32)

```
Base      Tier I    Tier II   Tier III  Tier IIII
 (1-g)     (h-p)     (q-t)     (u,v)      (w)
          h = 1+2   q = h+i   u = q+r   w = u+v
          i = 3+4   r = j+k   v = s+t
          j = 5+6   s = m+n
          k = 7+8   t = o+p
          m = 9+a
          n = b+c
          o = d+e
          p = f+g
```

Note: It's impossible for a mutation to reach `x` e.g. `w+x = x`.
